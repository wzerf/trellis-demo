Playwright 1.60+ coverage: on-demand HAR recording inside tracing, `locator.drop()`, page-level aria snapshot assertions, `test.abort()` from fixtures and route handlers, `getByRole({ description })`, `toHaveCSS({ pseudo })`, and `connectOverCDP({ noDefaults })` ‚ÄĒ plus all 1.59 features.

## Golden Rules

The patterns every guide assumes (full list with rationale in [SKILL.md](SKILL.md)):

1. **`getByRole()` over CSS/XPath** ‚ÄĒ resilient to markup changes
2. **Never `page.waitForTimeout()`** ‚ÄĒ use web-first assertions or `page.waitForURL()`
3. **Isolate every test** ‚ÄĒ no shared state, no order dependencies
4. **`baseURL` in config** ‚ÄĒ zero hardcoded URLs in tests
5. **Mock external services only** ‚ÄĒ never mock your own app
